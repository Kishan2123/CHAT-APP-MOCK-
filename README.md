Mock Chat Application

A simple and interactive chat interface built using HTML, CSS, and JavaScript that allows users to send and display messages locally in the browser without any backend. This project simulates a real chat experience using client-side technologies.

🚀 Features
Send messages using button or Enter key
Display messages in a chat-style interface
User vs Bot message distinction
Automated bot replies (mock responses)
Timestamps for each message
Delete messages functionality
Local Storage support (persists messages after refresh)
Responsive design (works on mobile & desktop)
Dark / Light mode toggle
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & responsive layout
JavaScript (Vanilla) – Logic & interactivity
Web Storage API – Data persistence

⚙️ How to Run

Download or clone the repository

git clone https://github.com/Kishan2123/CHAT-APP-MOCK-.git
Open the project folder
Double-click on index.html
The app will open in your browser
🧠 How It Works
User enters a message in the input field
JavaScript validates and displays the message
Message is stored in localStorage
A mock bot response is generated
Chat history is loaded automatically on refresh
🎯 Use Cases
Learning front-end development concepts
Understanding DOM manipulation & event handling
Demonstrating local storage usage
Prototype for building real-time chat apps
⚠️ Limitations
No real-time multi-user communication
No backend or database
Limited to predefined bot responses
Data stored only in browser
